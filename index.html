프로젝트 도구 : Android Studio, Github

[프로젝트 설명]
Java를 이용하여 앱을 제작한 첫 프로젝트입니다.
사용자의 Google Fit에 기록된 걸음 수 데이터와 같은 라이프로그 데이터, 사용자의 일정에 따른 걸음 예측 점수 데이터, 사용자가 원하는 운동 강도 데이터를 이용하여 그에 맞는 목표 걸음 수 및 맞춤형 헬스 컨텐츠를 제공하는 앱을 개발하는 프로젝트입니다.

이후 걸음 예측 점수와 사용자가 원하는 운동 강도 데이터를 이용하여 사용자의 다음주 요일별 최종 목표 걸음 수를 제공합니다.
사용자와 같은 나이대, 성별의 사용자들을 그룹화하여 매긴 주별 걸음 수 랭킹과 전체 사용자를 기준으로 매긴 주별 걸음 수 랭킹을 제공함으로써 사용자의 걷고 싶은 동기를 자극시킵니다.
사용자가 입력한 걸음 예측 점수에 맞춰서 그에 맞는 운동 영상 및 운동 자극 영상등을 제공합니다.
Frontend의 경우 oven을 이용하여 UI를 설계하고 그에 맞춰 앱의 UI를 구성했습니다. 또한 Google API를 이용하여 Google Fit의 데이터를 앱으로 가져왔고 이후 Backend로 보내 데이터를 DB에 저장했습니다.
Backend의 경우 Frontend로부터 받은 라이프로그 데이터를 DB에 저장하고 저장된 데이터들을 통해서 랭킹을 매기거나 prophet을 통한 걸음 수 예측을 했습니다.
